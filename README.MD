- 裏サイトのロゴ位置が揺らぐバグが残っているが解消できていない。
- 裏サイトはテンプレCSSにすでにレスポンシブ対応がなされているが、表サイトは不完全にしか行えていないのでこちらを完全なレスポンシブ対応にしたい。
